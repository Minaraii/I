    25. Rutina Semanal (6 Días)

    Bloque Fuerte (Lunes - Miércoles)

    • Lunes — Push + Estabilidad Superior: Push ups, Elevaciones laterales, Farmer's Carry
    • Martes — Pull + Postura: Pull ups, Farmer's Carry
    • Miércoles — Lower Body Fuerte: Sentadillas, Bisagra de cadera, Gemelos

    Bloque Suave / Volumen (Jueves - Sábado)

    • Jueves — Push Técnico: Push ups, Elevaciones laterales
    • Viernes — Pull Técnico: Pull ups, Farmer's Carry
    • Sábado — Lower + Core: Sentadillas ligeras, Gemelos, Farmer's Carry

    Domingo — Descanso Total

    Recuperación y movilidad ligera opcional.

    Fases de Entrenamiento

    • Alta intensidad: Lunes, Martes, Miércoles (prioridad alta)
    • Media intensidad / técnica: Jueves, Viernes, Sábado (prioridad media)
    • Baja intensidad / descanso: Domingo (prioridad baja)
